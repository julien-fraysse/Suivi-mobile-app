# Assets

Ce dossier contient les ressources statiques de l'application.

## Images requises

Vous devez ajouter les images suivantes dans ce dossier :

- `icon.png` (1024x1024) - Icône de l'application
- `splash.png` (1284x2778) - Écran de démarrage
- `adaptive-icon.png` (1024x1024) - Icône adaptative Android
- `favicon.png` (48x48) - Favicon pour le web
- `notification-icon.png` (96x96) - Icône pour les notifications

## Génération des assets

Vous pouvez utiliser des outils en ligne comme :
- [App Icon Generator](https://www.appicon.co/)
- [Expo Asset Generator](https://docs.expo.dev/guides/app-icons/)

Ou créer vos propres images en respectant les dimensions indiquées.
